# Scroll Indicator

A scroll indicator which shows scroll progress on page with visual cues to indicate page percentage coverage.

## Examples

This is a default implementation of the scroll indicator component. The scroll indicator in right-bottom shows the scroll progress and allows scrolling to top, and the progressing bar at the top of the screen shows you the similar progress just in a different way.
