# Tilted Carousel

A visually immersive carousel component with a 3D-tilted perspective and smooth bidirectional animations. The inspiration for this component comes from the Netflix landing page.

## Component Props

You can customize the behavior and appearance of the `TiltedCarousel` component using the following props.

### TiltedCarousel Props

  headers: ["Prop", "Type", "Default", "Description"],

    ["direction", "string (Optional)", "alternate", "Animation direction (left or right or alternate)."],
    ["pauseOnHover", "boolean (Optional)", "false", "Pause animation on hover."],
    ["preset", "string (Optional)", "default", "Preset configuration with custom values `compact`, `spacious`, `dramatic`."],

| Prop           | Type               | Default   | Description                                                                |
| -------------- | ------------------ | --------- | -------------------------------------------------------------------------- |
| images         | string[]           | []        | Array of image URLs to be displayed in the carousel.                       |
| className      | string (Optional)  | null      | Additional class names to apply to the carousel container.                 |
| rowClassName   | string (Optional)  | null      | Additional class names to apply to the row wrapper.                        |
| imageClassName | string (Optional)  | null      | Additional class names to apply to the image container.                    |
| speed          | number (Optional)  | 30        | Animation speed in seconds (lower is faster).                              |
| rows           | number (Optional)  | 4         | Number of horizontal rows in the carousel layout.                          |
| multiplier     | number (Optional)  | 6         | Number of times to duplicate the images array for seamless loop.           |
| direction      | string (Optional)  | alternate | Animation direction (left or right or alternate).                          |
| pauseOnHover   | boolean (Optional) | false     | Pause animation on hover.                                                  |
| preset         | string (Optional)  | default   | Preset configuration with custom values `compact`, `spacious`, `dramatic`. |
