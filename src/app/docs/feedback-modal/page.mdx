## Usage

You can customize the content of the feedback modal by passing the following props to the component.

### Feedback Modal

- `customId`: A unique identifier for the feedback modal.
- `title`: Optional title to be displayed in the feedback modal default is "Feedback".
- `className`: Additional class names to be applied to the feedback modal.
- `callback`: Optional callback function to be called when the submit button is clicked.

### Feedback Modal Props

- `customId`: A unique identifier for the feedback modal.
- `title`: Optional title to be displayed in the feedback modal default is "Feedback".
- `className`: Additional class names to be applied to the feedback modal.
- `callback`: Optional callback function to be called when the submit button is clicked.
- `children`: Optional content to be displayed inside the feedback modal.
- `...props`: Additional props to be passed to the feedback modal.

### Feedback Modal Content Props

- `textClassName`: Additional class names to be applied to the text area.
- `className`: Additional class names to be applied to the feedback modal content.
- `children`: Optional content to be displayed inside the feedback modal content.
- `...props`: Additional props to be passed to the feedback modal content.
