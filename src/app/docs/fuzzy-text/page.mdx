### Usage

You can use the `FuzzyText` component by passing the following props to it.

- `text`: (Optional) The text to be displayed. If not provided, the `orignal` text will be used.
- `orignal`: The original text to be displayed.
- `children`: The text to be displayed.
- `className`: Additional class names to be applied to the component.
- `delay`: (Optional) The delay in milliseconds between each character change. Default is 30.
