# Installation

<Steps>

<Step>Install Tailwind CSS</Step>

Components are styled using Tailwind CSS v4. Follow the [official installation guide](https://tailwindcss.com/docs/installation) to add it to your project. <br/>
If you are using React with Vite (TS), you can follow this [vite TS guide](https://ui.shadcn.com/docs/installation/vite) and for React (JS) follow this [vite JS guide](https://setup-shadcn-reactjs.hashnode.dev/step-by-step-guide-setting-up-shadcn-in-react-js-without-typescript).

<Step>Install Motion</Step>

Components are animated using [Motion](https://motion.dev/) check the required dependencies first.

<CodeBlock code={`npm install motion`} lang="bash" className="h-[52px]" />

<Step>Add the utility helper</Step>

Create a `lib/utils.ts` file to manage Tailwind CSS classes:

<CodeBlock filePath="./src/lib/utils.ts" lang="ts" className="h-fit" />

<Step>Install icons</Step>

Add [Lucide React](https://lucide.dev/) icons:

<CodeBlock code={`npm install lucide-react`} lang="bash" className="h-[52px]" />

<Step>That's it</Step>

You can now start adding components to your project.

<CodeBlock
  code={`npx shadcn@latest add "${srcUrl}/c/button.json"`}
  lang="bash"
  className="h-[52px]"
/>

The command above will add the Button component to your project. You can then import it anywhere in your project.

There is also option to add components manually via copying from the [Component Page](/docs/button/#installation).

</Steps>
